Linux networking commands

(1) ip (internet protocol)
e.g: ip a

(2) ping
e.g: ping google.com

(3) curl
e.g: curl https://sufiyan-khan-portfolio.lovable.app/

(4) wget webget
e.g: wget https://example.com/file.txt

(5) ss socket statistics
e.g: ss -tulnp 
flags: 
t > TCP
u > UDP
l > listening
u > numbers
p > process names

(6) netstat network statistics
e.g: netstat -tulnp
nowadays ss is used more 

(7) hostname
e.g: hostname 
output: sufiyan-khan-VirtualBox

(8) hostnamectl
e.g: hostnamectl
output : hostname + OS detailed info

(9) traceroute 
e.g: traceroute google.com

(10) nslookup
e.g: nslookup google.com

(11) ifconfig
e.g: ifconfig
better to use  ip a

(12) tracepath
e.g: tracepath google.com

(13) dig (domain information groper) 
e.g: dig google.com
